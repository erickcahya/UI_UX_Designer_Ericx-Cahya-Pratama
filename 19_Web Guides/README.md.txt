**Web Guides**

**Don’t Make Me (or Anyone) Think**

User ingin tau apa yang seharusnya mereka lakukan secepatnya dan langsung melakukannya.

**Design for Scanning, not reading**

Gunakan design yang konvensional, ciptakan hierarki visual, buat sesuatu yang clickable menjadi nyata dan jangan taruh semua detail dalam sekali klik.

**Time Wasting Sucks**
User online untuk menghemat waktu bukan untuk menghabiskan waktu.

**Let Me Go Back**
Kesalahan tidak perlu penting di mata user.

**People are Habitual**
Jika sesuatu berjalan dengan baik, user akan cenderung untuk tetap menggunakannya.

**Get to the Point**
Dengan penghematan waktu, user tidak akan mencari kepuasan di internet dan penting untuk menghapus kata atau kalimat filer.

**Some People Love Search**
Ada beberapa user yang memiliki sifat mencari sendiri

**Don’t Change the Map**
User mengingat pengalaman pertama berinteraksi dengan website dan aplikasi, persistent navigation, nama page, dan “you are here” navigation.

**Show Them the Way to Go Home**
Home Button menjadi salah satu jalan keluar untuk user.

**Don’t Argue but Test**
Testing dengan real user membuka dimensi baru disuksi dan juga membantu semuanya terlibat untuk mempelajari ulang tentang asumsi mereka.

**Increase your Reservoir of Goodwill**
Buat sesuatu hal yang penting tentang apa yang akan user lakukan di site anda dengan sangat mudah, ketahui informasi apa yang user ingin lihat / ketahui, dan save step wherever you can
